# Project Roadmap

## Overview

This roadmap outlines the planned evolution of AI Job Hunter. The project is being developed incrementally, with each version adding new functionality while maintaining stability and code quality.

---

# Version 1.2 (Current Release)

## Completed

### Job Collection

- Multi-source job scraping
- Government of Canada scraper
- Greenhouse scraper
- Lever scraper
- RemoteOK scraper
- Ashby scraper
- Adzuna scraper

### Data Processing

- Job validation
- Duplicate removal
- Canadian location filtering
- Job classification

### Resume Analysis

- Resume matching engine
- Skill extraction
- Resume compatibility scoring

### Scoring

- Rule-based scoring engine
- Weighted job ranking
- Company preference scoring
- Penalty rule system

### Data Storage

- SQLite database
- CSV export
- Daily Apply Today report

### Dashboard

- Streamlit dashboard
- Job statistics
- Top-ranked opportunities

# Long-Term Vision

The long-term goal is to transform AI Job Hunter into a complete job search assistant that helps users:

- Discover relevant opportunities
- Match jobs to their skills
- Prioritize applications
- Track application progress
- Improve resumes
- Prepare for interviews
