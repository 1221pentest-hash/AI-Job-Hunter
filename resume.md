# Resume Folder

## Purpose

This folder stores the resume used by the AI Job Hunter resume matching engine.

The application compares resume skills against job descriptions to calculate a compatibility score for each opportunity.

## Privacy

The author's personal resume is not included in this repository for privacy reasons.

To use the application:

1. Place your own resume in this folder.
2. Update the application configuration if necessary.
3. Run the job collection process.
4. Review the generated compatibility scores and reports.

## Supported Formats

Future versions are planned to support:

- PDF
- DOCX
- Plain Text

Current versions use structured resume data within the application.
